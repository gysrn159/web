CSS 특징

CSS는 HTML 문서에 있는 요소들에 선택적으로 스타일을 설정할 수 있는 언어입니다.

만약 HTML 페이지에서 <a> 태그가 적용된 모든 하이퍼텍스트를 빨간색으로 변경하려고 한다면 다음과 같이 CSS를 작성하면 됩니다.

<style>
  a {
    color:red;
  }
</style>

또는 아래와 같이 태그의 속성값으로 작성할 수도 있습니다.

<a href="1.html" style="color:red;">출력값</a>


위 예시와 같이 작성할 경우 아래와 같이 페이지에 출력될 겁니다.


CSS를 사용하여 위와 같이 웹 브라우저에 보이는 HTML 요소들에 스타일을 적용할 수 있습니다.


CSS의 구조를 'ruleset'이라고 합니다.


  • 선택자 (Selector)
    스타일을 적용하려고 하는 HTML 요소 이름. 다른 요소를 꾸미기 위해서는 선택자를 바꾸거나, 추가하여 사용할 수 있습니다.
  • 선언 (Declaration)
    속성과 속성값으로 구성된 하나의 규칙.
  • 속성 (Property)
    요소에 적용할 옵션의 종류.
  • 속성 값 (Property value)
    HTML 요소에 적용될 속성의 실제 값. 'color: red, blue, black' 등이 될 수 있습니다.
