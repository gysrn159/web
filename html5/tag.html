HTML 태그

HTML 태그 간단 소개

  • 태그(tag)란

    태그는 꺾쇠괄호(<>) 안에 둘러싸인 키워드이다. 꺾쇠괄호(<>) 안에 들어간 속성값에 따라 태그의 기능이 나뉩니다.

  • HTML 태그 기본 형식

    HTML 요소는 여는 태그와 닫는 태그 사이에 출력될 내용을 포함하고 있으며, 태그값에 따라 출력될 내용의 형태가 달라집니다.

     <tag>출력값</tag>

  • 주요 태그

    번호
    속성
    설명
    1
    <html>
    		 웹페이지의 시작과 끝.
    2
    <head>
    		 <title>, <meta> 등 웹 페이지의 정보, 외부 파일 링크 등을 정의.
    3
    <body>
    		 웹브라우저에 출력되는 내용.
    4
    <title>
    		 문서 제목 설정.
    5
    <meta>
    		 문서의 인코딩 정보를 정의.
    6
    <script>
    		 script 코드 삽입. 주로 Javascript 코드를 넣을 때 사용.
    7
    <div>,
    <span>
    		 요소들을 묶을 때 사용.
      ㄴ <div> - 블록 라인 요소(줄바꿈 O), <span> - 인라인 요소(줄바꿈 X).
    8
    <a>
    		 하이퍼링크를 지정할 때 사용.
    9
    <br>
    		 줄바꿈 태그.
    10
    <p>
    		 단락을 구분 짓는 태그.
    11
    <ul>,
    <ol>
    		 리스트 정렬.리스트 정렬.
      ㄴ <ul> : unordered list, 순서가 없는 리스트.
      ㄴ <ol> : ordered list, 순서가 있는 리스트.
    12
    <li>
    		 <ul>, <ol>의 항목을 나열할 때 사용.
