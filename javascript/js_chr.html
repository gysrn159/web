Javascript 특징

자바스크립트는 HTML과 CSS가 결합되고 웹페이지 상에서 올려진 후, 브라우저의 자바스크립트 엔진에 의해 실행됩니다. 이는 페이지의 구조와 스타일등을 정해놓고, 자바스크립트가 실행된다는 것과 같은 의미입니다.

동적으로 사용자 인터페이스를 업데이트하는 자바스크립트의 사용은 Document Object Model API를 통해 HTML과 CSS를 수정하는 것으로 좋은 현상입니다. 만약 자바 스크립트가 HTML과 CSS 전에 실행되었다면 문제가 분명 발생할 것입니다.


자바스크립트 문법

자바스크립트는 아래와 같이 작성됩니다.

<script>

 자바스크립트 코드

</script>

<body></body> 태그 안에 들어있는 <script></script> 사이에 자바스크립트 코드를 넣음으로서 웹 페이지 상에서 동작이 가능하게끔 할 수 있습니다.


또는 다음과 같이 외부에 있는 자바스크립트 파일을 불러올 수 있습니다.

<script src="script.js"></script>

웹 페이지에서 HTML 요소를 실행하는 중에 <script> 태그를 만나면 자바스크립트의 내용이 모두 실행되고 난 후에 다음 요소를 실행하기 때문에 위와 같이도 사용할 수 있습니다.
